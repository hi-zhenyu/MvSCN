## Multi-view Spectral Clustering Network

Simple implementation of our paper MvSCN.
The details can be found in the IJCAI2019 paper [here](https://www.ijcai.org/proceedings/2019/356).

### requirements
- Python 3.6
- scikit-learn
- tensorflow
- keras
- munkres

### run

```
python run.py
```
